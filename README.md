# Kartica Flashcards

Простое SPA для изучения слов с помощью флешкарточек. Приложение адаптировано под мобильные устройства, может быть установлено как PWA и поддерживает кастомизацию темы через Telegram WebApp API.

## Запуск локально

1. Запустите любой статический сервер из корня проекта, например:
   ```bash
   npx serve .
   ```
2. Откройте `http://localhost:3000` (или порт сервера) в мобильном браузере или эмуляторе.

Приложение автоматически зарегистрирует сервис-воркер и будет доступно для установки на рабочий стол.
