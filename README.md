# Kartica

Современное приложение на Vite + Vue для работы с карточками слов и интеграцией с Telegram Mini App.

## Требования
- Node.js 18+
- npm 9+

## Установка зависимостей
```bash
npm install
```

## Запуск в режиме разработки
Запустите Vite dev server и откройте [http://localhost:5173](http://localhost:5173):
```bash
npm run dev
```
По умолчанию Vite проксирует запросы `/api/*` на серверную заглушку, которую можно заменить реальным API.

## Сборка для продакшена
```bash
npm run build
```
Готовые статические файлы будут размещены в директории `dist/`. Для предпросмотра сборки используйте:
```bash
npm run preview
```

## Локальный API-заглушка
В директории `api/` находится обработчик `user.js`, возвращающий демо-данные Telegram-пользователя. Его можно адаптировать под реальный backend, используя валидацию `X-Telegram-Data`.
